# 🌐 Shaffof Xizmat

**AI-powered Civic Transparency Platform for public service accountability and legal awareness**

---

## 🚀 Overview

**Shaffof Xizmat** — bu fuqarolarga davlat xizmatlarini baholash, muammolarni ochiq bildirish va huquqiy savollarga AI orqali tez javob olish imkonini beradigan raqamli platforma.

Ilova maqsadi — fuqarolar va davlat xizmatlari o‘rtasida **shaffoflikni oshirish**, muammolarni ko‘rinadigan qilish va huquqiy savodxonlikni kuchaytirish.

---

## 💡 Key Features

### ⭐ Service Rating System

* Fuqarolar davlat xizmatlarini baholaydi (1–5 yulduz)
* Sharh qoldirish va tajriba bo‘lishish imkoniyati
* Eng muammoli xizmatlar aniqlanadi

---

### 🤖 AI Legal Assistant

* “Bu qonuniymi?” kabi savollarga oddiy tilda javob beradi
* Fuqaroga vaziyatni tushuntiradi
* Keyingi harakatlar bo‘yicha yo‘l-yo‘riq beradi

---

### ⚠️ Issue Reporting System

* Muammolarni anonim yoki ochiq tarzda bildirish
* Xizmat turi, izoh va dalil qo‘shish imkoniyati
* Real feedback tizimi shakllantiriladi

---

### 📊 Transparency Dashboard

* Xizmatlar statistikasi
* Eng ko‘p muammo bo‘lgan sohalar
* Reytinglar va tendensiyalar

---

## 🧠 Core Idea

Platforma 3 asosiy narsani birlashtiradi:

* 📊 Fuqarolar fikri (feedback)
* 🤖 AI huquqiy yordam
* 🏛 Shaffof xizmat nazorati

---

## 🛠 Tech Stack (reja sifatida)

**Frontend**

* Figma UI → React / Next.js (keyingi bosqich)
* Modern responsive design

**Backend (future)**

* Node.js / Python FastAPI
* AI integration (LLM-based chatbot)

**AI Layer**

* Legal Q&A assistant (Uzbek language support)

**Database**

* PostgreSQL / Firebase
